About

What is Randomize?

Randomize will generate random songs for people to try out! Songs are based on Spotify's Global Top 50 Charts, US Top 50 Charts, and songs users added to our database. Know a song you like that is not in the database? Submit them in Your Added Songs to be included for others to enjoy!

Introduction

Randomize is an application dedicated to generating random music recommendations for users to expand their musical horizons. It is a web application powered by HTML, Sequelize and Javascript. It is also formatted in CSS through the Bulma library.

Overview

Our application functions by pulling information from a mySQL database containing a list of the top 50 songs from Spotify. Our program utilizes an API created in Sequelize by our team to randomly select 10 songs from this database and place them in a randomized table at the userâ€™s request. Our application was developed for use in most modern internet desktop browsers and is compatible with Chrome, Firefox, and Edge.
